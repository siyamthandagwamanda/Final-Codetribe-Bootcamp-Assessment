Newsletter Subscription Website
Overview

This project is a responsive Newsletter Subscription Website developed using HTML, CSS, and JavaScript. The website allows users to subscribe to a newsletter by entering a valid email address. After successful submission, users receive a confirmation message on the screen.

The project was inspired by the provided design brief and was recreated using my own code and implementation while maintaining a similar visual appearance and user experience.

Features
Responsive desktop and mobile layout
Newsletter subscription form
Email validation using JavaScript
Error handling for invalid email addresses
Success confirmation message after subscription
Modern user interface with hover effects
Clean and structured code
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JavaScript)

How It Works
The user enters an email address.
JavaScript validates the email format.
If the email is invalid, an error message is displayed.
If the email is valid, a success message is shown.
The user can dismiss the success message and return to the form.
